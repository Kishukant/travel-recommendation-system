Travel Destination Recommendation System
This project recommends travel destinations based on user preferences such as:

Country
Season
Budget
Trip Type
Temperature
Rain Level
Duration
Crowd Level
Technologies Used
Python
Scikit-learn
Flask
HTML/CSS
Machine Learning
How to Run
Install dependencies
pip install -r requirements.txt

Run Flask app
python app.py

Open browser
http://127.0.0.1:5000

Features
Machine Learning recommendation model
Web interface using Flask
Top travel destination suggestions
