# Travel Destination Recommendation System

This project recommends travel destinations based on user preferences such as:

* Country
* Season
* Budget
* Trip Type
* Temperature
* Rain Level
* Duration
* Crowd Level

## Technologies Used

* Python
* Scikit-learn
* Flask
* HTML/CSS
* Machine Learning

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run Flask app

python app.py

3. Open browser

http://127.0.0.1:5000

## Features

* Machine Learning recommendation model
* Web interface using Flask
* Top travel destination suggestions
